# Auto-Parking Car Mobile App

A custom **Flutter mobile application** developed to control an intelligent auto-parking car via Bluetooth. The app provides smooth control, mode selection, and real-time feedback from the car.

---

## Features

### Motion Control
- Drive the car manually using intuitive on-screen controls.
- Supports forward, backward, left, and right movements.

### Mode Selection
- Switch between **Manual**, **Autonomous**, **Auto-Parking**, and **Path Learning** modes directly from the app.
- Displays the current active mode in real-time.

### Battery Monitoring
- Real-time battery status updates from the car.
- Visual alerts when the battery is low.

### Communication
- Bluetooth-based two-way communication.
- Sends commands to the car and receives sensor data for display.

### User Interface
- Clean and simple UI for easy operation.
- Smooth animations for buttons and sliders.
- Responsive layout suitable for different screen sizes.
